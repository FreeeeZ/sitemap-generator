Генератор Sitemap.xml

Инструкция:

Введите список URL-адресов (по одному на строку), которые нужно включить в sitemap.xml.

Вы можете указать дополнительные параметры для каждого URL (необязательно):

Формат: URL [частота изменений] [приоритет] [дата последнего изменения]

Пример: https://example.com/page1 monthly 0.8 2023-01-01
